Estructuras Condicionales Simples

Se puede observar en la gráfica una estructura condicional simple, donde el rombo indica y representa una condición, en esta gráfica hay dos opciones que se pueden tomar. Si la condición nos indica que es verdadero se sigue el camino de verdadero caso contrario por la izquierda que es falso.

Estructuras Condicionales Compuestas

Cuando se nos presente tomar alguna decisión tenemos que realizar una u otra actividad. Lo que quiero decir es que se puede tener actividades tanto por el lado verdadero o por el falso, lo mas importante es que se debe tomar en cuanta que se realiza NUNCA pero NUNCA se realiza ambas actividades falso y verdadero

Estructuras Condicionales Anidadas

Se tiene que decir que una estructura condicional es anidada , cuando de la rama del verdadero o falso se encuentra otra estructura condicional.

Ejercicio

Realizar un programa que ingrese por teclado tres notas de un estudiante, se pide calcular el promedio y tiene que imprimir los siguientes mensajes: si el promedio es >= 5 mostrar "aprobado" Si el promedio es >= 3 y < 4 mostrar "regular" Si el promedio es < 3 mostrar "reprobado"
